Research Areas

Our interdisciplinary team explores cutting-edge topics in cybersecurity, bridging the gap between academic research and real-world security challenges.

Offensive Security

Developing advanced penetration testing methodologies, red team operations, and attack simulation frameworks.

Vulnerability Research

Discovering and analyzing security vulnerabilities in modern software systems and emerging technologies.

Product Security

Designing secure-by-design principles and security assessment frameworks for product development lifecycles.

Threat Modeling

Creating comprehensive threat models and risk assessment methodologies for complex distributed systems.

Secure Development

Researching secure coding practices, static analysis tools, and automated security testing approaches.

Data Protection

Investigating privacy-preserving technologies, data anonymization techniques, and secure data handling practices.
